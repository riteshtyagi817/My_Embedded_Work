1. top
this commands shows processes usage
top --options as below
spacebar -- updates the display immediately
M - sort by current resident memory usage
T - sort by total CPU usage
P - sort by current CPU usage(default)
u - displays only one user's processes
f - select different statistics to display
? - displays a usage summary for all top commands
other commands like top -- htop and atop
